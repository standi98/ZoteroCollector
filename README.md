# Zotero Extracter
 
Automated script for extracting information from papers in Zotero.
 
## Quick Start
Make a '.env' file by copying '.env.template' and fill in the information. 'cp .env.template .env'
Run 'uv sync'
Run 'zotero_extract.py'


## General workflow
1. Import '(ZoteroTemplate)DataEntry' into zotero. Note that Zotero BetterNotes is a requirement.
2. Read papers and enter relevant information that you want to extract
3. Set up '.env' by copying '.env.template' and filling in information
4. Set up 'config.yaml' by entering the fields that you want to extract. 
5. Run 'zotero_extract.py'
